RPG Game in Java
A simple RPG (Role-Playing Game) developed in Java. This project showcases basic game mechanics, character interactions, and combat functionality. The game allows players to explore, battle enemies, and progress through various levels.

Features
Character Creation: Customize your character's name, attributes, and abilities.
Combat System: Engage in turn-based battles with enemies.
Level Progression: Advance through levels and improve your character's stats.
Inventory Management: Collect items and manage inventory to enhance your character's capabilities.
Requirements
Java 8 or higher
